# Assumptions

- ASSUMPTION-01: The application is web-based.
- ASSUMPTION-02: Every user must authenticate before accessing the system.
- ASSUMPTION-03: Role-based access control is required for lecturer, student, and administrator roles.
- ASSUMPTION-04: Assignment submissions require file upload.
- ASSUMPTION-05: Students can view grades and feedback only after the lecturer publishes them.
- ASSUMPTION-06: Administrators can manage users and course data but cannot change grades.
- ASSUMPTION-07: The system records important actions for audit and troubleshooting.
