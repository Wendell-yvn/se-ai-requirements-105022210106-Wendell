# Interview Answers

## Lecturer Interview Summary

1. Lecturers want to create assignments by course.
2. Lecturers want students to see deadline information clearly.
3. Lecturers need to download or view submitted files.
4. Lecturers want to enter grades and written feedback.
5. Lecturers need a list of students who have not submitted.

## Student Interview Summary

1. Students want a dashboard showing active assignments.
2. Students want deadline reminders or visible deadline status.
3. Students want confirmation that their file was submitted successfully.
4. Students want to view grading status, grades, and feedback.
5. Students worry about upload failure near the deadline.

## Administrator Interview Summary

1. Administrators need to manage users and roles.
2. Administrators need to manage course information.
3. Administrators need to assign lecturers and students to courses.
4. Administrators need basic reports about assignment and submission activity.
5. Administrators want configuration options for submission rules.
